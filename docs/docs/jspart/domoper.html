DOM操作

xframework延用了Zepto的选择器，可以像使用Zepto一样使用它，但它没有提供对ajax的封装，关于数据请求，你可以自己封装一个类库或者使用其他类库，更详细的文档请参阅Zepto官网或者国内Zepto资源站核心部分(文档核心部分中的after和before方法在xframework中不存在了，用来代替其的是insertAfter和insertBefore)。
