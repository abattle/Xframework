概述

在原Zepto的基础上，扩展了一些用于类型判断和数组转换等的工具方法。

isNil

检查value是否是null或者undefined。

isUndefined

检查value是否是undefined。

isNull

检查value是否是null。

isNaN

检查value是否是NaN。

isArguments

检查value是否是arguments对象。

isElement

检查value是否是DOM元素。

isBoolean

检查value是否是布尔值。

isTypedArray

检查value是否是系统数组而不是类数组。

isDate

检查value是否是日期类型。

isDocument

检查value是否是文档对象。

isString

检查value是否是字符串类型。

isNumber

检查value是否是数字类型。

isSafeInteger

检查value是否是安全数字类型。

isEmpty

检查value是否是空对象,集合,map或者Set。

isEqual

比较两个值是否相等，值可以是Plain Object或者基本数据类型值。

dataset

获取传入对象上以"data-"开头的属性。

getTranslate

获取元素css3的translate属性。

parseJSON

将对象字符串转化为JSON。

stringify

将JSON对象转为对象字符串

hasVal

判断给定plain object是否有给定属性值：$.hasVal(obj,key);

hasVal

判断给定plain object是否有给定属性值：$.hasVal(obj,key);

formatNumber

数字补0操作给定方法：$.formatNumber(num,n)。n默认为2。

fixStatusBar

fixStatusBar让顶部栏具有沉浸式效果。

getRandomID

得到一段指定长度的随机数字字符串。

numToChinese

将阿拉伯数字转化成中文数字表示。

random

产生一个指定范围内的随机数。
