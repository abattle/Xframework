介绍

    xframework是一套结合了Zepto，Framework7等开源框架优点而写成的一套框架，适合用于开发Hybird App应用，在修复了一些开源框架在低版本android手机的兼容性问题后，xframework已经能支持到android4.2+以及ios7+的手机。当然，它只是一个最初版本，如果你有兴趣，请用你的智慧来和我一起完善它吧。
